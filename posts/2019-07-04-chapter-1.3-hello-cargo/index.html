Create new project

The command create a project directory, 2 files Cargo.toml, and src/main.ts. It also initialize the direction as a git repository.

1

$ cargo new hello_cargo

Check project

cargo check check a local package and all of its dependencies for errors

1

$ cargo check

Build project

cargo build compile a local package and all of its dependencies

1
2
3
4
5

# build debug
$ cargo build

# build release
$ cargo build --release

Run project

cargo run run a binary or example of the local package.

1
2

4

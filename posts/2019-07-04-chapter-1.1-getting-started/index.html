This is my notes of learning The Rust Book from a C++ programmer’s perspective, it is recommended to read the book online or offline by command rustup doc.

Install Rust

Install Rust using rustup.

1

$ curl https://sh.rustup.rs -sSf | sh

After installation, running rustc —version to check the installed version.

1
2

$ rustc --version
rustc 1.36.0 (a53f9df32 2019-07-03)

Update installation by running rustup update, for more information see the rustup documentation.

In the Rust development environment, all tools are installed to the ~/.cargo/bindirectory, and this is where you will find the Rust toolchain, including rustc, cargo, and rustup.

To uninstall rust and rustup, run rustup self uninstall.

Learn Rust

Learn rust with the official documentations. The documentation is also available locally using the rustup doc command.

  • Read the book
  • Do the rustlings course
  • Check out rust by examples
